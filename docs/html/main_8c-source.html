00025 \section definition Definicion
00026 El problema de nesting se puede definir como encontrar una dispocision de
00027 patrones que esten dentro de otro de forma que se maximice el area utilizada.
00028 Este problema tiene muchas variantes, pero en el proyecto se trabajara
00029 especificamente sobre el caso de Knapsack.
00030 
00031 \section objective Objetivo
00032 Encontrar una dispocicion de patrones que maximicen el area
00033 utilizada dentro de otro patron. Los patrones estan definidos como poligonos
00034 simples, los cuales pueden ser por definicion convexos o concavos pero solo
00035 pueden tener un adentro o mas claramente sus lineas no se intersectan.
00036 
00037 \section case Caso de Estudio
00038 Aunque el problema es teorico, el projecto quiere generar una aplicacion
00039 que pueda ser utilizada por la industria marroquinera, donde tienen que
00040 definir como distribuir los moldes de corte dentro de un cuero, en este caso
00041 el cuero se puede definir como el patron o poligono donde se tienen que
00042 colocar los demas patrones, y los moldes como los patrones internos.
00043 */
